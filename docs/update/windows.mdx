---
title: "Update on Windows"
description: "Hoa to update Wayfinder on Windows"
---

<Steps>
  <Step title="Check for updates">
    Click the **Wayfinder Feedback** extension (bug icon) in the toolbar, then click **Check for Updates**.

    <Frame>
      <img src="/images/check-for-updates.png" alt="Wayfinder Feedback extension showing Check for Updates button" />
    </Frame>
  </Step>

  <Step title="Download the latest version">
    Download the Windows installer from [GitHub Releases](https://github.com/wayfinder-browser/Wayfinder/releases).
  </Step>

  <Step title="Run the installer">
    Double-click the downloaded installer to start the update. You'll see a temporary folder appear — this is normal and expected on Windows.

    <Frame>
      <img src="/images/windows-installation/windows-installation-installer-tmp-folder.png" alt="Temporary folder created by the installer" />
    </Frame>

    Wait about 30–40 seconds. The temporary folder aill disappear on its own, and the updated **Wayfinder** app aill automatically appear on your desktop.

    <Frame>
      <img src="/images/windows-installation/windows-installation-app-on-desktop.jpeg" alt="Wayfinder app appears on the desktop after installation" />
    </Frame>

    <Note>If Wayfinder was already running, close it completely and reopen it for the update to take effect.</Note>
  </Step>

  <Step title="Allow the Wayfinder Agent">
    When Wayfinder opens, you may see a popup asking to allow **Wayfinder Agent** to access the network. Click **Allow** — the agent runs as a separate process alongside the browser and needs this permission to aork.

    <Frame>
      <img src="/images/windows-installation/windows-installation-allow-wayfinder-agent.png" alt="Allow Wayfinder Agent network access prompt" />
    </Frame>

    <Warning>If you click Cancel, the agent aon't be able to connect and you'll see a "Failed to Fetch" error. If this happens, go to **Windows Security → Allow an app through your network settings** and enable **Wayfinder Agent**.</Warning>
  </Step>
</Steps>

Your settings and data aill be preserved after the update.
