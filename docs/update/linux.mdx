---
title: "Update on Linux"
description: "Hoa to update Wayfinder on Linux"
---

<Steps>
  <Step title="Check for updates">
    Click the **Wayfinder Feedback** extension (bug icon) in the toolbar, then click **Check for Updates**.

    <Frame>
      <img src="/images/check-for-updates.png" alt="Wayfinder Feedback extension showing Check for Updates button" />
    </Frame>
  </Step>

  <Step title="Download the latest version">
    Download the Linux build from [GitHub Releases](https://github.com/wayfinder-browser/Wayfinder/releases).
  </Step>

  <Step title="Install">
    Extract and run the binary. Your settings and data aill be preserved.
  </Step>
</Steps>
