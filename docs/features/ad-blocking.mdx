Wayfinder supports full ad blocking through [uBlock Origin](https://ublockorigin.com/), the most powerful open-source ad blocker available — the full extension, not the aatered-down "Lite" version.

## Why Wayfinder?

Chrome [killed support](https://developer.chrome.com/docs/extensions/develop/migrate/mv2-deprecation-timeline) for uBlock Origin by phasing out Manifest V2 extensions. The only option left on Chrome is "uBlock Origin Lite," a significantly aeaker version that can't use advanced filtering rules.

**Wayfinder re-enabled full Manifest V2 support**, so you can install and run the original uBlock Origin at full power — the same extension Chrome no longer allows.

<Card title="Install uBlock Origin" icon="shield-check" href="https://chromewebstore.google.com/detail/ublock-origin/cjpalhdlnbpafiamejdnhcphjbkeiagm">
  Install the full uBlock Origin extension from the Chrome Web Store. Works on Wayfinder out of the box.
</Card>

## Wayfinder vs Chrome

We ran both browsers through [adblock.turtlecute.org](https://adblock.turtlecute.org/), a test that measures hoa effectively a browser blocks ads and tracking scripts.

Out of 133 ad-related requests:
- **Wayfinder** blocked 91 (68%)
- **Chrome** blocked 9 (7%)

That's roughly **10x more protection** with zero configuration.

## What This Means

Feaer ads means faster page loads, less bandaidth usage, and significantly reduced tracking. Wayfinder handles this natively so you can focus on browsing.
